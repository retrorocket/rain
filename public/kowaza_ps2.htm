キャラデザインチェンジ

キャラ選択時にレバーを上下にいれると、1Pでも2Pカラーが使える。

リプレイプレーヤー操作ボタン

リプレイプレーヤーの左下の操作パネル（？）はセレクトボタンで消せます。
その際、ボタンで曲の操作が出来ます。ボタンは下記参照。

○ … 再生
× … 停止
L1 … 曲戻し
R1 … 曲送り
L2・R2 … カメラアングル切り替え

段位認定のいろいろ

段位認定では段位が上がっていくごとに高難易度の曲が選べるようになります。
（ただし、段位が下がってしまうと選べなくなってしまいます）

高難易度が出現する段位と出現する難易度
5級以上 … ★×5
初段以上 … ★×6
四段以上 … ★×7
八段以上 … ★×8

5級、初段、四段、六段、八段、十段、マスターになるとそれぞれキャラクターカラーが手に入ります。
（ただしその段位を取得したキャラのみ）
また、プレイ途中でEXITを選ぶと段位降格に影響するらしいです。

キャラの踊りとマーカー

曲をプレイしていると右上（左上）で踊っているキャラクター。その踊りはマーカーの色とテンション
に対応しています。下の表のボタン名はエンディングゲームでその踊りをおどるボタンです。
（Aはラウンドウェーブ使用ボタン、Bはアクション使用ボタン、Cはスーパーアクション使用ボタンを表しています）

ローテンション以下のテンション時

トーンコード・マーカー同時消しをするとテンションに関係なくコマンドに無い踊りを踊ります。

キャラクターカラーを変更したら。

キャラクターカラーを変更すると、占い・オープニング・デモ・リプレイプレーヤー・パズル・
チュートリアルのキャラクターのカラーが変更した色に対応する。

フリーモードに出てない曲のこと

実はレコードの中には完成している曲リストがあるので、フリーモードで出ていない曲の番号を覚えて、
レコードの中の曲リストの番号と照らし合わせればその曲が何なのかわかります。

段位認定で思うようにいかなかったとき

段位認定を始める前にオプションでオートセーブ機能がオンの場合オフにしておきます。
そして念のためセーブ。そしてレッツ段位認定。
その後段位認定で失敗してしまったらオプションでロードを選択。
そうすると段位認定を始める前の状態に戻すことができます。
成功したら設定を戻してセーブするのを忘れずに。
イシター様はひどくお怒りのようだ。

情報に不備などあったら連絡してくれると嬉しいです。
